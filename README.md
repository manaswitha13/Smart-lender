# Smart Lender – Applicant Credibility Prediction for Loan Approval

## Project Overview
Smart Lender is a machine learning-based web application that predicts whether a loan application is likely to be approved based on applicant information. The system helps financial institutions make faster and more accurate loan approval decisions.

## Features
- Loan eligibility prediction
- User-friendly web interface
- Flask-based backend
- Random Forest machine learning model
- Data preprocessing using StandardScaler
- Instant prediction results

## How to Run
1. Clone the repository.
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the Flask application:
   ```
   python app.py
   ```
4. Open your browser and visit:
   ```
   http://127.0.0.1:5000/
   ```

## Machine Learning Model
- Algorithm: Random Forest Classifier
- Preprocessing: StandardScaler
- Dataset: Loan Prediction Dataset
